# mynode
learn node.js with koa and vue
clone 仓库后,$ npm install安装node_modules
$ node app.js运行项目，http:localhost:3000可以进行访问
项目集成了koa.js,vue.js,sequelize,nunjucks
数据库使用本地mysql数据库test，在配置文件config.js配置
